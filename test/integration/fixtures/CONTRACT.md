# Integration Test Fixture Contract

All servers implement identical endpoints for cross-implementation testing.

## Endpoints

| Method | Path | Behavior | Status |
|--------|------|----------|--------|
| GET | `/healthz` | Returns `"ok"` | 200 |
| GET | `/` | Returns server greeting | 200 |
| ANY | `/echo` | Echoes request body, headers in `X-Echo-Headers` | 200 |
| GET | `/large?bytes=N` | N-byte zero-filled body | 200 |
| GET | `/status/{code}` | Returns HTTP status `{code}` | {code} |
| GET | `/delay?ms=N` | Sleeps N ms, then returns | 200 |
| GET | `/chunked` | Streams 100 × 1KB chunks with 10ms delay | 200 |
| GET | `/gzip` | 100KB body with `Content-Encoding: gzip` | 200 |
| GET | `/trailers` | Response with trailer headers | 200 |
| GET | `/never` | Never responds (for cancel tests) | — |

## Certificates

Self-signed CA + server cert generated by `scripts/gen-certs.sh`:
- CN=localhost
- SAN: localhost, nginx, undertow, nghttp2, 127.0.0.1, 172.30.0.{10,11,12}
- Validity: 365 days
- Location: `fixtures/certs/server.pem`, `server.key`, `ca.pem`

## Transport modes

| Server | h2 (TLS+ALPN) | h2c (cleartext) | Notes |
|--------|:---:|:---:|-------|
| Go net/http | ✅ | ✅ | In-process, no Docker |
| nginx | ✅ | ❌ | nginx rejects h2c cleartext |
| Undertow | ✅ | ✅ | Separate listeners |
| nghttpx | ✅ | ✅ | Separate frontends |
